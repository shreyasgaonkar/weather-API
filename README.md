# Weather App

Replace ```sample.config.js``` to ```config.js``` and replace:

```
var config = {
    API_ID: 'ENTER-API-ID',
    IPSTACK_API_ID: 'ENTER-API-ID'
};

```


### Uses
- [OpenWeatherMap API](api.openweathermap.org)
- [Scalable IP Address Geolocation & Reverse IP Lookup REST API](https://apilayer.com/)
